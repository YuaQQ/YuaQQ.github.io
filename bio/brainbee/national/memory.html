学习与记忆
Learning and Memory

信息加工模型

  • 编码（Encoding）：新信息获取 → 巩固（Consolidation）
  • 存储（Storage）：通过神经系统的持久变化存储
  • 提取（Retrieval）：访问并使用存储的信息

学习类型

  • 刺激-反应学习（Stimulus-response learning）：
    • 经典条件反射、操作性条件反射
    • 赫布法则：突触前与突触后神经元同时反复激活会加强该突触
  • 运动学习（Motor learning）：在感觉引导下建立运动系统变化（如骑自行车）
  • 知觉学习（Perceptual learning）：学习识别先前感知过的刺激（如物体识别）
  • 关系学习（Relational learning）：学习刺激间的关系（如物体相对位置）
  • 情景学习（Episodic learning）：记忆事件的顺序

记忆类型

  • 感觉记忆（Sensory memory）：短暂保留原始感觉（如回声记忆）
  • 短期/工作记忆（Short-term/Working memory）：容量有限（约7±2项），由前额叶皮层协调
  • 长期记忆（Long-term memory）：由海马和海马旁区巩固
    • 非陈述性/内隐/程序性记忆：技能和条件反射记忆（如系鞋带）
    • 陈述性/外显记忆：可言语化的记忆
      • 语义记忆：事实性知识（如骨骼名称）
      • 情景记忆：事件及其情境记忆（如考试经历）

记忆相关脑区

  • 大脑皮层：感觉记忆
  • 非陈述性记忆：
    • 小脑：运动控制与协调
    • 基底神经节：习惯中心
  • 陈述性记忆：
    • 齿状回：事件记忆
    • 海马体：记忆形成与提取
      • 位置细胞：对特定位置记忆激活，参与导航
      • 网格细胞：位于内嗅皮层，在无地标时追踪空间位置
  • 前额叶皮层：工作记忆、长期存储与巩固

长时记忆编码涉及的变化

  • 突触数量与形态
  • 神经递质释放量
  • 突触后膜受体数量
  • 陈述性记忆在海马编码，后转移至额叶长期存储与巩固

神经可塑性（Neuroplasticity）

  • 神经系统在出生后被修饰的能力
  • 突触可塑性：
    • 长时程增强（LTP）：谷氨酸系统突触变化，增强突触后细胞EPSPs（在杏仁核和海马）
      • 机制：NMDAR激活 → Ca²⁺内流 → AMPAR插入突触后膜 → CREB磷酸化 → 合成可塑性蛋白
    • 长时程抑制（LTD）：降低活跃突触效能
      • 机制：长时间低强度谷氨酸激活 → 低水平Ca²⁺内流 → 磷酸酶激活 → AMPAR内吞 → 突触消除
  • 成年神经发生：
    • 神经干细胞位于：
      • 海马结构的颗粒下层
      • 侧脑室旁的室管膜下区
    • 新神经元功能：
      • 室管膜下区 → 迁移至嗅球（嗅觉）或纹状体（运动控制与认知）
      • 颗粒下层 → 形成中间神经元，辅助学习记忆
